# SC2002 Internship Placement Management System CLI

## Project Directory Structure

### Source Code
All Java source code (`.java` files) is located in the `src/` directory, organised by package:

`src/Main.java`

`src/boundary/`

`src/controller/`

`src/entity/`

`src/utility/`

### Data Files

All CSV data files are located in:

`src/data/`

### Javadoc

The generated Javadoc HTML documentation is located in the `doc/` folder (you may download `doc.zip` instead if you wish).

To view the documentation, open `doc/index.html`
